# Youth Passion Project Landing Page

A revamped landing page for the Youth Passion Project, built with ReactJS and styled using the official YPP design palette.

## Features

- Responsive design
- Modern UI with gradient backgrounds
- Clean typography using Playfair Display and DM Sans fonts
- Color scheme based on YPP brand guidelines
- Sections for About, Programs, and Contact

## Design Palette

The design incorporates the official Youth Passion Project color palette:
- Deep Purple (#3B0F6E)
- Primary Purple (#6B21C8)
- Mid Purple (#8B3FE8)
- Light Purple (#B47FFF)
- Lavender (#E8D8FF)
- Blush (#F3ECFF)
- White (#FFFFFF)
- Ink (#1A0533)
- Muted (#6B5F7A)

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm

### Installation

1. Clone the repository
2. Navigate to the project directory: `cd ypp-landing-page`
3. Install dependencies: `npm install`

### Running the App

`npm start`

Runs the app in development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### Building for Production

`npm run build`

Builds the app for production to the `build` folder.

### Testing

`npm test`

Launches the test runner.

## Deployment

The built files in the `build` folder can be deployed to any static hosting service like Netlify, Vercel, or GitHub Pages.
